## ¿Qué es una Pila?
Es una estructura de datos lineal, la cual nos sirve para guardar datos de una forma escalonada, que funciona con el principio de LIFO (Last In, First Out)

## ¿Qué es una Cola?
Es una estructura de datos lineal, la cual nos sirve para guardar datos de forma lineal, que funciona con el principio de FIFO (First In, First Out)
